API Usage

API URL - relist.axorax.repl.co


Revolt user information

GET /user/:user_id
Example Description
/user/01G...7XN Get all data
/user/01G...7XN?data=createdAt,username Get specific data

Revolt server information

GET /server/:server_id
Example Description
/server/01G...AHH Get all data
/server/01G...AHH?data=discoverable,ownerId,owner Get specific data

Vote for a bot

GET /vote?id=author_id&bot=bot_id
Example Description
/vote?id=01G...7XN&bot=01G...QTX Vote for a bot

Submit a bot

GET /submit-bot?id=author_id&bot=bot_id&prefix=bot_prefix
Example Description
/submit-bot?id=01G...7XN&bot=01G...QTX&prefix=r! Submit a bot

Login

GET /requestlogin?id=author_id
Example Description
/requestlogin?id=01G...7XN Issue a login request
After creating a login request, the server will give you a token. The user must copy and paste `r!login <token>` in Revolt, in any server that has the "Relist" bot.

Check login token verification

GET /checklogin/:login_token
Example Description
/checklogin/Wei28Wpd Check if user login token is verified
After the token was created with requestlogin and the user pasted it, the system will verify the token. So, you can use checklogin and it will respond with either true or false.

Botlist

GET /botlist
Example Description
/ Bot list information
/range/1/10 Bots in a range (sorted from most voted)
/ids All bot ids in list
/raw Raw JSON data
/raw/:bot_id Raw JSON data for a certain bot
/search/:query Search for a bot with id or name
/:bot_id Information on a certain bot
/:bot_id/owner Information about the owner of a bot
/usernames All bot usernames in list
/tag/:tag_name All bots with a certain tag
/random One random bot from list
/random?amount=10 Random bots from list
